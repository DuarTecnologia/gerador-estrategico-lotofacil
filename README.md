# 🎯 Gerador Estratégico Lotofácil

Sistema profissional de geração estatística de jogos para Lotofácil.

## ✨ Funcionalidades

- ✅ Geração inteligente baseada em análise estatística
- ✅ 5 estratégias diferentes (Balanceada, Quentes, Frias, Pares/Ímpares, Sequencial)
- ✅ Classificação de dezenas em Quentes/Neutras/Frias
- ✅ Simulação de desempenho
- ✅ Interface responsiva e moderna
- ✅ Sistema de autenticação com Supabase
- ✅ Planos: Mensal (R$27) e Vitalício (R$97)

## 🚀 Acesso

- **Página inicial:** `/index.html`
- **Checkout:** `/checkout.html`
- **Login:** `/login.html`
- **Dashboard (protegido):** `/dashboard.html`

## 🛠️ Tecnologias

- HTML5 / CSS3 / TailwindCSS
- JavaScript (ES6+)
- Supabase (Autenticação)
- Font Awesome Icons
